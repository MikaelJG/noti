BEM (Block Element Modifier)

Block is a card: class="card"
Element is a photo, p, div: class="card_picture" - "_" designates it's a child
Modifier is changing something additional on a block or an element: class="card_button card_button--active" -->"--active" is the modifier

.card {
    padding: 8px;

    &_img {
        display: block; // Modifying the element "card_img"
        &--active {
            color: yellow; //Targetting the modifier "card_img--active"
        }
    }
}
